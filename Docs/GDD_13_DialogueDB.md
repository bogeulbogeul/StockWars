# StockWars GDD: 13. 안나 다이얼로그 데이터베이스 (Dialogue DB)

**버전:** v2.37.0 (최종 완결본 - 안건 상황별 대사 전수 데이터베이스화)  
**기능:** 안나(Anna)의 상황별 대사 ID, 발동 조건 및 연출 명세

---

## 0. 디자인 의도 (Design Intent)
본 데이터베이스는 유니티(Unity) 개발 단계에서 JSON 혹은 CSV로 즉시 변환 가능한 구조를 지향합니다. 유저의 게임 데이터(자산, 수익률, 신뢰도 등)와 실시간으로 연동되어 안나의 풍부한 리액션을 구현함으로써, 트레이딩의 기계적인 지루함을 '캐릭터와 함께하는 서사'로 승화시킵니다.

## 2. 대사 연출 가이드 (UX/Interaction Guide)

### 2.1. 텍스트 출력 방식
- **타이핑 효과 (Typewriter Effect)**: 기계적인 느낌을 주기 위해 1글자당 0.05초의 딜레이로 텍스트 출력.
- **키워드 강조**: 종목명, 수익률, 골드 액수는 특정 색상(예: 상승-Red, 하락-Blue)으로 하이라이팅.

### 2.2. 초상화 연동 (Portrait Animation)
- 대사의 `ID`와 `Emotion` 태그에 따라 화면 하단 안나의 픽셀 아트 초상화가 즉시 전환됨.
- **Standard**: 무표정하지만 차분한 시선.
- **Happy**: 입가에 부드러운 미소, 눈이 살짝 가늘어짐.
- **Pain**: 눈썹을 찌푸리며 고개를 약간 숙임.
- **Angry**: 시선이 날카로워지며 팔짱을 끼는 자세.

---

## 3. 확장 가능성 (Technical Scalability)
- **멀티어 (Multi-Language)**: 동일 ID에 언어 코드(`KR`, `EN`)만 추가하여 글로벌 현지화 대응 가능.
- **랜덤 배리에이션**: 동일한 상황(예: ANN_WELCOME)에 3개 이상의 대사를 배치하여 유저가 반복되는 대사에 지루함을 느끼지 않도록 처리.
