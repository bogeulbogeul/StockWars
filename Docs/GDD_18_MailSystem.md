# StockWars GDD: 18. 스마트폰 메일 시스템 (Mail System)

**버전:** v1.0.0 (초안 구축)  
**핵심 컨셉:** Lo-Fi 스마트폰 앱을 통한 비동기적 정보 및 재화 수령 시스템  

---

## 1. 시스템 개요 (System Overview)
유저는 게임 내 스마트폰 UI의 **'Mail' 앱**을 통해 시스템 알림, 친구의 선물, 시장의 주요 소식을 확인하고 수령할 수 있습니다. 이는 유저가 오프라인일 때 발생한 사건들을 피드백하는 핵심 창구로 기능합니다.

---

## 2. 메일 데이터 구조 (Mail Data Structure)

| 필드명 | 타입 | 설명 |
| :--- | :--- | :--- |
| **MailID** | string | 각 우편의 고유 식별자 (GUID) |
| **Type** | Enum | System, Social, Market, Event |
| **Sender** | string | 발신자명 (NPC 이름 혹은 친구 유저네임) |
| **Title** | string | 우편함 리스트에 노출될 제목 |
| **Content** | string | 메일 상세 본문 (Rich Text 지원) |
| **Reward** | Item/Gold | 수령 가능한 아이템 혹은 재화 데이터 |
| **ExpiryDate** | DateTime | 만료 시점 (TTL: 기본 7일) |
| **IsRead** | bool | 읽음 여부 상태 체크 |

---

## 3. 카테고리별 세부 내용

### 3.1. 시스템 메일 (System)
- **발신자**: Node Finance, 안나(Anna), 시스템 관리자
- **내용**: 
    - 주간 이자 정산 요약 리포트.
    - 자산 압류(Seizure) 전 최종 경고 알림.
    - 튜토리얼 완료 보상 및 이벤트 상금.

### 3.2. 소셜 메일 (Social)
- **발신자**: 친구 목록 내 유저명
- **내용**:
    - 친구가 보낸 **스테미너(Energy) 선물**.
    - 친구가 공유한 **찌라시 복사본** (클릭 시 인벤토리로 이동).

### 3.3. 시장 메일 (Market)
- **발신자**: Cipher Securities, 시장 분석국
- **내용**:
    - **신규 상장(IPO)** 예고 및 개시 알림.
    - 보유 종목의 **상장 폐지(Delisting)** 및 정리 매매 개시 통보.

---

## 4. UI/UX 연출 가이드

### 4.1. 스마트폰 앱 인터페이스
- **아이콘**: 픽셀 아트로 그려진 빈티지한 봉투 모양 아이콘.
- **배지 (Badge)**: 읽지 않은 메일이 있을 경우 스마트폰 홈 화면 앱 아이콘 우측 상단에 빨간색 pips 노출.
- **알림 사운드**: 드로이드 풍의 낮은 톤 비프음 (Lo-Fi 감성).

### 4.2. 주요 기능 인터페이스
- **일괄 수령 (Collect All)**: 하단에 고정된 버튼으로 모든 [Social] 선물을 한 번에 수령 및 삭제.
- **삭제 (Delete)**: 클릭한 메일 우측 상단의 'X' 버튼으로 개별 삭제. 읽은 메일은 자동으로 투명도가 낮아집니다.

---

## 5. 기술적 고려 사항
- **Asynchronous Data**: 유저가 오프라인일 때 서버(혹은 백엔드 로직)에서 생성된 메일 정보를 로그인 시점에 데이터 동기화.
- **TTL (Time To Live)**: `ExpiryDate`가 경과한 데이터는 DB 및 클라이언트 리스트에서 즉시 소멸 처리하여 데이터 무결성 유지.
