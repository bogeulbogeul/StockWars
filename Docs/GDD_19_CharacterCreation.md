# StockWars GDD: 19. 캐릭터 생성 시스템 (Character Creation)

**버전:** v1.0.0 (초안 구축)  
**핵심 컨셉:** 사이퍼 증권(Cipher Securities) 트레이더 등록 및 출입증 발급 연출  

---

## 1. 시스템 개요 (System Overview)
유저는 게임을 처음 시작할 때 자신의 가상 아바타와 투자 성향을 설정합니다. 이는 Lo-Fi 감성의 빈티지한 키오스크 혹은 단말기 UI 형태를 띠며, 완료 시 유저의 정보가 담긴 '트레이더 출입증'이 출력되는 연출과 함께 게임 본편(홈 오피스)으로 진입합니다.

---

## 2. 캐릭터 커스텀 (Initial Appearance)

### 2.1. 헤어 스타일 (Hair)
유저는 4종의 기초 헤어 스타일 중 하나를 선택할 수 있습니다.
- **숏컷 (Short)**: 깔끔하고 활동적인 픽셀 스타일.
- **롱 (Long)**: 차분하고 부드러운 느낌의 긴 머리.
- **포니테일 (Ponytail)**: 묶은 머리 형태의 스포티한 스타일.
- **단발 (Bob-cut)**: 세련된 단발 스타일.

### 2.2. 고정 의상 (Fixed Outfit)
초기 몰입도를 높이기 위해 기본 의상은 다음과 같이 고정 제공됩니다.
- **상의**: 베이직 화이트 티셔츠 (White T)
- **하의**: 베이직 블랙 슬랙스 (Black Pants)

### 2.3. 랜덤 생성 (Randomize)
- UI 내 '랜덤(Random)' 버튼 클릭 시, 4종의 헤어 스타일 중 하나가 무작위로 선택되어 프리뷰에 반영됩니다.

---

## 3. 투자자 성향 선택 (Investment Background)

유저는 4대 핵심 능력치 중 하나에 보너스(+1 블록)를 주는 성향을 선택합니다. 이는 초기 게임 플레이의 유리한 지점을 결정합니다.

| 성향명 | 영문명 | 상징 블록 | 효과 포인트 |
| :--- | :--- | :--- | :--- |
| **예리한 분석가** | Analyst | **분석력** | 찌라시 가독성 소폭 상승 및 노이즈 필터링 강화 |
| **베테랑 협상가** | Negotiator | **협상력** | 초기 대출 이자율 감면 및 매매 수수료 추가 할인 |
| **공격적 자산가** | Manager | **운용력** | 보유 가능 종목 수 +1 및 종목별 매수 한도 증대 |
| **불굴의 트레이더** | Resilient | **회복력** | 스테미너 회복 속도 보너스 및 알바 보상 추가 |

---

## 4. UI/UX 및 연출 가이드

### 4.1. 화면 구성
- **좌측**: 아바타 실시간 프리뷰 공간. 픽셀 아트 애니메이션(숨쉬기 등) 노출.
- **우측**: 설정 패널 (이름 입력창, 헤어 선택 버튼, 성향 선택 카드).
- **하단**: '등록 완료(Register)' 버튼.

### 4.2. 이름 설정 및 검증
- **길이**: 2~8자 (한글/영문/숫자).
- **중복 체크**: 로컬 저장 데이터 및 서버 데이터와 연동하여 중복 여부 확인.

### 4.3. 완료 연출
1. 유저가 '등록 완료' 클릭.
2. 도트 프린터 소리와 함께 화면 하단에서 '트레이더 출입증' 이미지가 스르르 출력됨.
3. 출입증에는 유저가 설정한 아바타 얼굴, 이름, 성향 명칭이 픽셀로 인쇄되어 있음.
4. "Welcome to StockWars, [Name]." 문구와 함께 홈 오피스 씬으로 전환.
